# IsshoHub 簡介

**IsshoHub**（一緒 Hub）是一個專為香港／台灣人移居日本而設的雙語資訊平台。

## 定位

「在日生活，一緒開始」——涵蓋從簽證申請到日常生活的全方位指南，讀者群主要是正在考慮或已經移居日本的繁體中文使用者。

## 技術架構

- **前端**：純靜態 HTML + 原生 CSS + Vanilla JS，部署在 Cloudflare Pages
- **後端**：Supabase（PostgreSQL）儲存文章、用戶收藏、熱門搜尋等
- **CMS**：自製輕量編輯器（嵌入在前端頁面，登入後可見）
- **語言切換**：繁中 / 英文，全站 i18n 由 `IsshoCore` 管理
- **地震速報**：串接 JMA Atom feed，即時顯示在頂部 topbar

## 內容分類

| 分類 | 說明 |
|------|------|
| 新聞資訊 | 日本時事、政策、生活相關新聞 |
| 簽證・在留資格 | 各類簽證申請、在留政策 |
| 創業・工作 | 在日創業、求職 |
| 住屋 | 租屋、買樓、物業資訊 |
| 文化 | 日本文化、生活習慣 |
| 稅務・保險・年金 | 稅務、社保相關 |
| 日常手續 | 駕照換領、銀行、SIM 卡等 |
| 好去處 | 旅遊、休閒 |
| 寵物 | 帶寵物赴日相關資訊 |
| 人物故事 | 移居者的真實故事 |

## 頁面結構（首頁）

1. 頂部 topbar（地震速報）
2. 導覽列（分類 + 語言切換）
3. Hero 主焦點文章 + 側欄 3 篇
4. 編輯精選（水平卡片，支援 video 標記）
5. 新聞資訊（3 篇）
6. 分類瀏覽
7. 人物故事（可橫滑）
8. 最新文章
9. 電子報訂閱
