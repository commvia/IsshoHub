# 高度專門職積分試算器 設計規格

**目標：** 在 IsshoHub visa 頁面的高度專門職區塊內，嵌入互動式積分試算器，讓用戶根據日本官方積分表自行估算分數。

**架構：** 純前端 Vanilla JS，無後端，嵌入現有 `visa/index.html`，積分邏輯抽離為獨立 JS 檔案 `assets/js/hsp-calculator.js`，樣式追加至 `styles.css`。

**技術棧：** HTML + CSS + Vanilla JS，雙語（繁中／英文，跟隨 `IsshoCore` 語言切換）。

---

## 積分路徑

三條路徑，用戶先選路徑，再填對應問題：

| 代號 | 路徑 | 適用對象 |
|------|------|---------|
| `academic` | 學術研究 (i)(a) | 研究員、教授 |
| `technical` | 高度技術 (i)(b) | IT、工程師、專業人士 |
| `business` | 高度經營 (i)(c) | 高階管理人員 |

門檻：**70 分**以上方可申請高度専門職 (i)。

---

## UI 結構

```
[路徑選擇器] — 三個分段按鈕

[問題區 — 左/主欄]          [即時分數 — 右欄]
  ① 學歷                      目前得分
  ② 職業經歷                    XX 分
  ③ 年收入（含年齡分層）         門檻 70 分
  ④ 年齡                      進度條
  ⑤ 研究業績（部分路徑）
  ⑥ 特別加算（多選）
                               [查看結果]

[結果區 — 點擊後展開]
  總分 + 各項細分表
  達標狀態（✓ 達標 / ✗ 未達標）
  建議（差幾分、哪些項目可提升）
```

手機版：右欄分數改為頁面底部固定 bar（`position: sticky`）。

## 結果區邏輯

```
總分 = 學歷 + 職業 + 年收入 + 年齡 + 研究業績 + 特別加算 + 獨有項目

if 總分 >= 70:
  顯示「✓ 預計達標」
elif 總分 >= 60:
  顯示「距達標差 X 分，建議考慮…」
else:
  顯示「距達標差 X 分，以下項目可提升分數…」

建議邏輯（差距 > 0 時）：
  - 若未選日語加分 → 提示 N2/N1
  - 若年齡 < 40 且未得年齡最高分 → 提示年齡已得分
  - 若未選日本大學 → 提示
  - 若職業經歷有提升空間 → 提示
```
